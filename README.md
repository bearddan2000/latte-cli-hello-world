# latte-cli-hello-world

## Description
A POC for latte programming language.

## Tech stack
- gradle
- java
- groovy
- python

## Docker stack
- openjdk:jdk8
- rzuckerm/latte

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
